Vision-Based Models for Sorting and Segregation of Waste

[ Code ]   [ Slides ]   [ Demo ]  

This work was completed during an internship at Omdena. We developed object classification and detection-based models to segregate trash into common categories.We also created a new dataset containing over 16000 images of various trash items upon which we tested multiple architectures of Deep Nets and SSDs and achieved an overall accuracy of 96.14% for 10-class classification.Finally, the comprehensive solution was presented as a proof-of-concept to an UN-SDG India team.
Tech Saplings: Intelligent Environment and Plant Health Monitoring and Evaluation System

[ Code ]   [ Slides ]  

We propose an end-to-end system for monitoring the plant's environmental conditions and health, including a range of embedded sensors for measuring conductivity, pH levels, soil moisture, etc. in addition to a drone-based camera system for counting and imaging data. This data is fed into object identification, segmentation, and prediction models in order to forecast illnesses, drooping, fire and other health condition evaluations.We found that this low-cost system is an effective solution for inspecting forests and other plantation initiatives. This project was awarded 1st prize at the 1st prize at the Smart Odisha Hackathon-2022.
Pancreatic 3D-Segmentation using CT Scans 🔗

In this work, we developed a transformer architecture for 3D medical image segmentation using an encoder-decoder architecture with linear complexity based on a pre- exisiting study. We evaluated this approach on multiple CT pancreatic segmentation datasets for cancer diagnosis.We achieved an overall accuracy of 91.32% while using fewer annotations to deliver highly feasible segmentation performance and precise uncertainty quantification.

Satellite Data based Pollution Forecasting using CNNs 🔗

This study explores the viability and reliability of using convolutional neural networks on satellite images to predict the BreezoMeter Air Quality Index (BAQI). We presented a new BAQI index dataset containing over 10000 satellite images and respective BreezoMeter air quality data from 57 cities. We explored multiple deep CNN architectures with variations in layers and data augmentation techniques and achieved an overall accuracy of 87.14%. We find that satellite images, which are inexpensive and ubiquitous,are effective in predicting air pollution.

CoughVid: Covid-19 Detection from Cough Voice Samples 🔗

This project was created as part of the Pfizer Digital Medicine Challenge. We developed an RNN and MFCC feature-extraction based model to detect Covid-19 from coughing audio samples.In real-world testing with over 320 patients, we obtained an overall accuracy of 82.42%. In addition, we discovered that MFCC and Spectrograms of audio signals can be utilized as image datasets to build CNN Models to categorize audio samples.
